# comptalib-test-backend

## Project setup
```
yarn install
```

### start dev
```
yarn dev ( dev mode )
yarn start
```

## Endpoints

### Company
```
GET /societies
GET /society/{id}
POST /society
PUT /society/{id}
DELETE /society/{id}
POST /user/add-company

```

### User
```
GET /users
GET /user/{id}
POST /user
PUT /user/{id}
DELETE /user/{id}
POST /company/add-user

```
